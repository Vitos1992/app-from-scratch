JavaScript

JavaScript (JS) — це високо рівнева, інтерпретована мова програмування, що використовується для створення інтерактивних веб-сторінок, динамічного контенту (анімації, форм), веб-додатків та навіть серверного бекенду через Node.js, дозволяючи браузерам додавати функціональність без перезавантаження сторінки.

JavaScript

HTML (HyperText Markup Language) — це стандартна мова розмітки, яка використовується для створення структури та вмісту веб-сторінок, визначаючи, як браузер відображає текст, зображення, посилання, списки, таблиці та інші елементи, формуючи скелет будь-якого сайту.

JavaScript

CSS (Cascading Style Sheets — каскадні таблиці стилів) — це мова стилів, яка використовується для оформлення зовнішнього вигляду вебсторінок, створених за допомогою HTML.

Git (Гіт) — це найпопулярніша розподілена система керування версіями, створена Лінусом Торвальдсом у 2005 році для відстеження змін у вихідному коді програм.

JavaScript

React (або React.js/ReactJS) — це популярна відкрита JavaScript-бібліотека для створення інтерфейсів користувача (UI) у вебзастосунках, розроблена та підтримувана компанією Meta (Facebook).
